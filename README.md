# Flappy-Bird

I created a simple game named "Flappy Bird" using HTML,CSS and Javascript. In this game the player controls a character that keeps falling down and player must jump using the X, Spacebar or Arrow up key to keep the character in the air while passing through gaps between continuous pipes. I added a score counter that increases by one for each pipe passed. If the player collides with a pipe the game ends and "GAME OVER" message is displayed .The game can be restarted by pressing any jump key.
I haven't used any Ai tools while making this project
